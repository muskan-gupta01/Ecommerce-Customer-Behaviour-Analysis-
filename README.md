# Ecommerce-Customer-Behaviour-Analysis-
E-commerce Customer Behaviour Analysis is a project that analyzes customer purchasing patterns and online shopping behavior to generate insights for improving sales, marketing strategies, and customer experience.
# IIoT Customer Behavior Analytics Project

## Project Overview

This project focuses on analyzing customer behavior and online shopping patterns using Python Data Analytics libraries. The dataset was cleaned, processed, analyzed, and visualized to identify customer trends, purchasing behavior, conversion patterns, and business insights.

The project was implemented using:

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

# Work Done in the Project

## 1. Data Cleaning and Preprocessing

The following preprocessing tasks were performed:

* Changed Customer ID data type
* Fixed Purchased column values
* Cleaned numeric columns
* Replaced missing (NaN) values using mean values
* Handled missing Region values
* Normalized numerical data
* Detected outliers using Z-score method

---

## 2. Statistical Analysis

Different statistical operations were performed on the dataset:

* Calculated mean, median, and standard deviation
* Found maximum and minimum values
* Identified rows where values were greater than a threshold
* Calculated variance
* Computed correlation coefficient matrix

---

## 3. Feature Analysis

Several important customer-related features were analyzed:

* Average Order Value
* Conversion Rate
* Loyalty Score
* Return Customer analysis
* Items Added to Cart
* Pages Viewed
* Review Rating
* Weekend vs Weekday purchasing behavior

---

## 5. Customer Behavior Analysis

The project also included detailed customer behavior analysis:

* Purchase patterns based on age and location
* Discount impact on order value
* Weekend and weekday shopping comparison
* Cart item behavior analysis
* Customer engagement analysis using pages viewed and time spent
* Conversion analysis using shopping activity data

---

## 6. Business Insights Generated

From the analysis, the following business insights were identified:

* Customers with higher loyalty scores generally place higher-value orders.
* More time spent on the website can increase order value.
* Discounts influence customer purchasing behavior.
* Conversion rates vary depending on customer activity.
* Product categories and device types affect customer engagement.
* Weekend shopping behavior differs from weekday behavior.
